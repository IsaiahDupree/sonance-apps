About

Grab the exact color of any pixel on your screen. Press a hotkey, use the magnifier loupe, and the color is instantly on your clipboard — as HEX, RGB, HSL, or ready-to-paste code.

GREAT FOR
• Matching a color from any app, site, or image
• Copying a HEX value straight into your design tool
• Pasting a SwiftUI Color or NSColor line right into code (Pro)
• Checking text contrast for accessibility

Uses the native macOS eyedropper, so it needs no special permissions and lands on the exact pixel every time.

FEATURES
• System eyedropper with a magnifier loupe
• HEX / RGB / CSS + WCAG contrast ratio (free)
• SwiftUI / NSColor / HSL code formats (Pro)
• Palette history of everything you've picked (Pro)

Free: pick + HEX/RGB/CSS. Pro (one-time purchase): code formats + palette history.
